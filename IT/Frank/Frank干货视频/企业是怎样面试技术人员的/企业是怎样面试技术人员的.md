# 企业是怎样面试技术人员的

## 1. 开发测试人员的评测标准：

1. 素质结构：
   - 事业心：你对你未来的这份工作（程序员）是怎么看的。
     你就不能诚实的回答：掉头发，996，没有女朋友啊这些。情商高点。如果你说你需要一份高薪的工作，如果是小企业可能会要你，但是大企业是不会要的。
   - 战略观念：比如：你要去面试微软云 Azure，它可能会问你你怎么看带未来的云发展这些问题，你需要在网上去查一下，微软云在这么多云中的定位，优势等。哪怕别人认为这个岗位未来没有前景，你自己要认为它很有前景，你是有备而来的，这时候面试官就会对你另眼相看。
   - 开拓性
   - 协作性
   - 进取心
   - 坚韧性：创业型的企业不会考虑你的事业心，战略观念，只看你能不能加班，就这么简单。
   - 诚实性

2. 智力结构：
   - 基础理论知识是否扎实（如：前后端：数据结构与算法） 
   - 专业知识（你做过哪些东西）
   - 观察力（找 bug）
   - 判断力

3. 能力结构：
   - 发现问题的能力
   - 获得信息的能力
   - 创新能力

4. 绩效结构：
   - 技术经济效益

后两个是近年来有的观念。面试大企业的时候重要的是前四点，尤为重要。

## 2. 技术管理人员的评测：（十年以上，比较屌的那种）

1. 素质结构：
   - 法制观念
   - 事业心
   - 技术和经济概念
   - 责任心

2. 智力结构：
   - 专业知识
   - 知识面
   - 对新技术新产品的敏感性
   - 思维力（周密性）

3. 能力结构：
   - 科学技术的鉴别能力 -> 所用的技术是否适合这个项目
   - 灵活性 -> 要留有余地
   - 信息沟通能力
   - 协调能力

4. 绩效结构
   - 科学技术的成果
   - 社会经济效益

## 3. 题外话

### 开放性问题：

如果有些大企业问非常开放性的问题，如微软有几次面试：你怎么估计美国加油站的数量；你怎么国际密西西比河的流量等问题，这时候，它看重的是你的**分析问题的能力**，因为这种问题本身没有标准答案，只有美国政府最后统计才知道。

### 解决：

很多大公司会问你对公司的看法，可以去网上搜一搜各**大公司的使命**。

### 创业公司：

#### 其实你面试也是在挑选公司：

面试的时候不止是公司挑你，你也要会挑公司，看它的水准。★

有些公司最后一个问题：你有什么想问我的吗？这种问题就是该你问问题了，不要问你们是干什么的。。

#### 不能去的公司：

你可以问**贵公司的核心产品，它的竞争优势在哪**，如果这个问题它答不上来，那这家公司你就不用进了。

1. 因为该公司极可能**在夹缝中生存**：“产品极度依赖于一些平台”，它只能跟着一个平台走，它迟早倒闭。
2. 接外包

> 这样的公司没有任何前景，你学不到任何东西，也不能愉快的合作，开发的产品也是一塌糊涂。

#### 继续问问题：

接下来你们将要承载什么样的团体；贵公司接下来想要发展成怎样的文化 / 想要塑造的企业文化是什么 / 您对贵公司的企业文化是怎样看待的。

#### 进阶问题：（小米公司举例）

贵公司这么多产品线，您认为那种将来最具有竞争力。

如果它回答不上来，拐了个弯子，说：这不是我管，我是管技术的。那你也拐个弯子，问它一些技术问题，如：拿着该公司的介绍单子，说为什么该公司在这个产品上要采用这个技术，我来谈谈该技术的优点和缺点，问他你们是不是也这样想的。

接下来，如果你们谈的非常愉快，就可以考虑谈谈别的：

工作时间 / 工作是否弹性，有没有什么娱乐项目、公司的福利，最后再谈薪资。

#### 一些坑：

如果面试官不给你介绍自己的职位的话，那么该公司肯定有问题。这能看出该公司的待客之道，如果它不尊重面试者，那么该公司忙起来的时候肯定一团糟。

如果那个面试官觉得自己是搞技术的，跟公司企业文化没关系，前两个问题都没大上来，那你可以直接走了，就不用面试了，拎着包就走。当面试官不尊重面试者的时候，自己被心情所左右的时候，这个公司是没有前途的。

### 大型公司：

大型公司就不用问企业文化了，因为它们已经定格了，可以在网上搜到；否则它们会认为你对技术圈不懂。★

可以直接问福利待遇，比较现实。或者其它的你关注的问题，只要你们谈得开心，就 OK 。

可以问你们团队是怎么样的，氛围如何。

如果它是你的头儿，你可以问他一些生活状态：喜不喜欢音乐。。

#### 禁忌：

不要问一些隐私，如：别人的私生活和工作经验（您过去在哪高就），这些都是禁忌。

## 4. 实例：

Frank：我还记得，前苹果首席工业设计师负责人乔纳森在90年代末时候说的一些话，这些话是这样的。
乔布斯问：“这家公司已经破败不堪了，这不是我的苹果，它没品味、没格调、没设计，所以除非他们把你绑在这个地方，否则你为什么还在这（指着乔纳森）？”
乔纳森：“这个嘛，是因为我还相信苹果的理念，也是你的理念，乔布斯”
乔布斯：“那你说我的理念是什么？”
乔纳森：“我认为，你（指乔布斯）始终相信，所谓的电脑，无论是什么产品，应该都是个人自然的延伸，正是那种使命感，那种对质量、对理想和心灵的专注让我们选择留下，让我们期待着再创辉煌。” 乔布斯：“好的，无论你正在做什么，今天剩下的时间就别想它了，我要大家去设计一个新的东西，我不管它是什么，只管放手去做，做个有用的东西，你叫什么名字？” 乔纳森：“Jony Ive！Steve，Welcome back.”

如果他的回答不是这样，是之前十年前的程序员的回答（“字体不重要”），估计乔布斯上任就会把他开了，可以看出，企业文化素养有多么重要

另外在第一节里讲的创新能力，是适用于苹果，而并非其他企业，并不是那个人不听乔布斯的话 ，这么理解你就太OUT了，是因为乔布斯早就说好的，我们要创新，字体也要，但那个人就是不理解为什么要在字体上创新，他从本质上就不愿意去做这件事情。为什么会有人理解为：“这和创新没关系呢？”
