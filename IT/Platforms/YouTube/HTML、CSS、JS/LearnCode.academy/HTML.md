# HTML

## 常用的盒子标签（分层）

`<div></div>`：divide 。

`<nav></nav>`：navigation 。

`<figure></figure>`：图片盒子。内可包含`<figcaption></figcaption>`，来对图片进行说明。

### 网页分层

一个网页通常分为四层：`<header></header>`、`<section></section>` 和 `<footer></footer>`
